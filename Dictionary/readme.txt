This Assignment is the implementation of Hash Table for the development of a special kind of dictionary.
This dictionary supports various functions mentioned below:
1. Addition of a word.
2. Updation of a word.
3. Deletion of a word.
4. Spell-Check: This function checks if a word is present in the dictionary or not if present then return the meaning of the word else returns the the list of nearest correct word/spelling if the word is not in the dictionary.
